# OFDM Image Transmission using MATLAB

## Overview
This project implements an Orthogonal Frequency Division Multiplexing (OFDM) system for image transmission using MATLAB. The system evaluates transmission performance over AWGN and Rayleigh fading channels using BER and PSNR metrics.

## Features
- OFDM modulation and demodulation
- Adaptive modulation
- Image transmission
- AWGN channel simulation
- Rayleigh fading channel simulation
- BER vs SNR analysis
- PSNR vs SNR analysis
- Received image reconstruction

## How to Run
1. Open `ofdm.m` in MATLAB.
2. Run the script.
3. Observe the transmitted and received images.
4. Analyze the BER and PSNR performance over different SNR values.
