# OFDM Image Transmission using MATLAB

## Overview
This project implements an Orthogonal Frequency Division Multiplexing (OFDM) system in MATLAB for image transmission over wireless communication channels.

## Features
- OFDM modulation and demodulation
- Image transmission
- AWGN channel
- Rayleigh fading channel
- BER vs SNR analysis
- PSNR vs SNR analysis
- Received image reconstruction

## Tools Used
- MATLAB

## How to Run
1. Open `ofdm.m` in MATLAB.
2. Run the script.
3. Observe the BER, PSNR, and received image results.
